####运行iOS
* 打开终端,切换到将用来保存项目的目录
* checkout
 ```git clone https://github.com/zqmReactNative/react-native-redux-marry.git```
* 切换到项目根目录
 ```cd react-native-redux-marry```
* 加载依赖包(Node建议v6.0.0+)
```npm install```
* 找到`ios`目录，打开`Marry.xcodeproj`
